Sinopsis Aplikasi 
Aplikasi ini adalah aplikasi yang akan membantu dan memudahkan user dalam belajar membuat karya seni. 
Di awal tampilan aplikasi akan ada 3 menu untuk menentukan pembuatan karya berdasarkan bahan yang ada. 
1. Origami, membuat karya menggunakan kertas origami seperti karya burung bangau menggunakan kertas origami. 2. Barang Bekas, membuat karya menggunakan barang yang sudah tidak terpakai seperti botol, kock bulutangkis, atau kain perca 3. Lainnya, membuat karya menggunakan barang yang baru, seperti kain flannel, tali kur dsb. Manfaat : - Menyediakan tutorial karya seni untuk pembelajaran setiap orang. - Memudahkan user belajar membuat suatu karya seni. - Mengajarkan user membuat suatu karya berharga dari barang bekas Pengguna : 
- Anak-anak, - Ibu Rumah Tangga, - dll
